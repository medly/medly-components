## Getting Started

```properties
npx create-react-app medly-components-demo --typescript
cd medly-components-demo
yarn add styled-components @types/styled-components
yarn add @medly-components/theme @medly-components/utils @medly-components/icons @medly-components/core @medly-components/layout @medly-components/loaders @medly-components/forms
```

Replace content of `index.tsx` with the following:

```tsx
import { defaultTheme } from '@medly-components/theme';
import { CssBaseline } from '@medly-components/core';
import { ThemeProvider } from 'styled-components';
import React from 'react';
import * as ReactDOM from 'react-dom';
import App from './App';

ReactDOM.render(
    <ThemeProvider theme={defaultTheme}>
        <CssBaseline />
        <App />
    </ThemeProvider>,
    document.getElementById('root')
);
```

Replace content in `App.tsx` with the following:

```tsx
import React from 'react';
import { Button } from '@medly-components/core';

const App: React.FC = () => <Button variant="solid">Click me!</Button>;

export default App;
```

If you are using a static property of any component like `Modal.Header`, you must add `"strictNullChecks": false` in your `tsconfig.json` file.

## Packages

Library consists of the following five packages:

### [Core](https://github.com/medly/medly-components/tree/master/packages/core)

The Core package consits of almost all basic react components like Avatar, Button, Input, List, Modal, etc.

**Example**:

<Preview>
    <List variant="horizontal">
        <Button variant="outlined">Outlined Button</Button>
        <Button variant="solid">Solid Button</Button>
        <Button variant="flat">Flat Button</Button>
    </List>
</Preview>

### [Icons](https://github.com/medly/medly-components/tree/master/packages/icons)

The Icons package consists of most commonly used svg icons exposed as react components.

**Example**:

<Preview>
    <EditIcon size="M" />
</Preview>

### [Loaders](https://github.com/medly/medly-components/tree/master/packages/loaders)

The Loaders package consists of some simple svg loaders exposed as react components.

<Preview>
    <ConcentricCircleLoader size="M" />
</Preview>

### [Forms](https://github.com/medly/medly-components/tree/master/packages/forms)

Use the Forms package to create dynamic forms using simple json.

<Preview>
    <Form
        fieldSchema={{ email: { type: 'email', label: 'Email' }, password: { type: 'password', label: 'Password' } }}
        actionLabel="Login"
        onSubmit={action('Submitted')}
    />
</Preview>

### [Layout](https://github.com/medly/medly-components/tree/master/packages/layout)
Use the Layout package to create page layouts using side panels and navigation items, which are exposed as react components.


**Example**:

```tsx
<SideNav>
    <SideNav.NavList>
        <SideNav.NavItem>
            <SideNav.NavIcon>
                <HomeIcon />
            </SideNav.NavIcon>
            <SideNav.NavText>Home</SideNav.NavText>
        </SideNav.NavItem>
    </SideNav.NavList>
</SideNav>
```

### [Theme](https://github.com/medly/medly-components/tree/master/packages/theme)

Style components across websites using the Theme package. Theme can either be applied to a particular type of component (Example: Making all icons solid) or to the entire website (Example: Changing the color theme of all type of components across the website).

### [Utils](https://github.com/medly/medly-components/tree/master/packages/utils)

The Utils package consists of the most commonly used functionalities and components. Eg [String Helper](https://github.com/medly/medly-components/tree/master/packages/utils/src/stringHelpers.ts) has functions to convert a string to camelCase, case insensitive string comparison, etc.
