# Typography

Use 'Typography' to design your content efficiently. We use the 'Open Sans' typeface by default, however, you can alter the font theme.

## How to use?

You must include a styled component global style using `CssBaseline` component like, seen below. `CssBaseline` component will add all the font faces and default font style.  After that, you may use the `Text` component with various configuration to match desired output and HTML tag.

```tsx
<ThemeProvider theme={defaultTheme}>
    <CssBaseline />
    <Text textVariant="h1" textWeight="Strong">
</ThemeProvider>
```

## Font Variants

Use font variant to decide the font style: (`font-size`,`font-weight`,`line-height`,`letter-spacing`) and `html tag` as well. If you have used `textWeight` prop as `Strong`|`ExtraStrong`, then both `body1` and `body2` variant will use the `strong` HTML tag.

| Variant | HTML Tag |
| ------- | -------- |
| h1      | h1       |
| h2      | h2       |
| h3      | h3       |
| h4      | h4       |
| h5      | h5       |
| body1   | span     |
| body2   | span     |
| body3   | span     |
| button1 | span     |
| button2 | span     |
