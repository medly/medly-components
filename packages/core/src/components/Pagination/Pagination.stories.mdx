# Pagination

The `Pagination` component allows you to show the data in a sequence of pages and select a specific page from a range of pages.

<Preview withToolbar>
    <Story name="Pagination">
        {() => {
            const [activePage, setActivePage] = useState(1);
            return (
                <DummyWrapper>
                    <Pagination
                        activePage={activePage}
                        totalItems={number('Total Items', 1000)}
                        itemsPerPage={number('Items per page', 20)}
                        hidePrevNextLinks={boolean('Hide Prev Next Links', false)}
                        onPageClick={setActivePage}
                    />
                </DummyWrapper>
            );
        }}
    </Story>
</Preview>
