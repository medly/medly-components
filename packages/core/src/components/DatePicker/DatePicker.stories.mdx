# DatePicker component

It is a controlled component and its state must handled.

```tsx
const [date, setDate] = useState<Date | null>(null);
return <DatePicker label="Select Date" value={date} onChange={setDate} />;
```

You can either enter the date manually or use the calendar inside the datepicker component. To open the calendar, click on the calendar icon inside the datepicker component, and to close the calendar, select any date or click outside the component.

### Validations

1. Validations happen on invalid and blur events.

```tsx
const FormWithDatePicker: React.FC = () => {
    const [date, setDate] = useState<Date | null>(null),
        handleSubmit = (e: React.FormEvent) => e.preventDefault();
    return (
        <form onSubmit={handleSubmit}>
            <div>
                <DatePicker value={date} onChange={setDate} label="Select Date" required />
            </div>
            <Button type="submit">Submit</Button>
        </form>
    );
};
```

<Preview withToolbar>
    <FormWithDatePicker />
</Preview>

2. Validations happen only on blur events when you either enter an incorrect value or focus out of the component.

```tsx
const DatePicker: React.FC = () => {
    return <DatePicker value={null} onChange={() => {}} label="Select Date" />;
};
```
