# CssBaseline

The `CssBaseline` is a global style component that provides an elegant, consistent, and simple baseline to build upon. It must reflect in the main tree of your application.

It contains base resets and font links used by the library.

```tsx
<ThemeProvider>
    <CssBaseline />
    ...
</ThemeProvider>
```
