# SearchBox

### Add a auto suggest text box

<Preview withToolbar>
    <Story name="SearchBox">
        {() => {
            const [options, setOptions] = useState([]),
                onInputChangeHandler = value => value.length > 2 && setOptions([...optionsArray, { label: 'Test', value: 'Test' }]);
            return (
                <SearchBox
                    placeholder="Search"
                    onInputChange={onInputChangeHandler}
                    onOptionSelected={action('Selected')}
                    options={options}
                    size={select('Size', size, 'M')}
                    customSearchFilter={boolean('Show Custom Search Filter', false)}
                    showSearchFieldShadow={boolean('Show Search Field Shadow', true)}
                />
            );
        }}
    </Story>

### Keyboard Support

It also supports keyboard navigation when it is in focus

| Key        | Function                           |
| ---------- | ---------------------------------- |
| Down Arrow | Moves focus to the next option     |
| Up Arrow   | Moves focus to the previous option |
| Enter      | Select the focused option          |
