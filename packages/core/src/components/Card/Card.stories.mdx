# Card


A simple card component with a lot of customization. The hover effect will work only when you pass the `onClick` prop. You can play with the props in the **Canvas** tab. 

<Preview withToolbar>
    <Story name="Card" decorators={[withKnobs]}>
        <Card
            variant={select('Variant', variants, 'solid')}
            flowDirection={select('Flow Direction', flowDirection, 'horizontal')}
            alignItems={select('Items Position', itemsPosition, 'center')}
            display={select('Display', display, 'flex')}
            separator={boolean('Separator', true)}
            withoutPadding={boolean('Without Padding', true)}
            fullWidth={boolean('Full Width', false)}
            fullHeight={boolean('Full Height', false)}
            minWidth={text('Min Width', '')}
            minHeight={text('Min Height', '')}
            onClick={action('Clicked')}
        >
            <Card variant="flat" display="flex" alignItems="center">
                <Text textWeight="Strong" textVariant="h2">
                    8
                </Text>
                <Text>Marks</Text>
            </Card>
            <Card variant="flat" display="flex" flex={6} alignItems="left">
                <Text>17607124</Text>
                <Text textVariant="h3">John, Snow</Text>
                <Text>Son of Lyanna Stark and Rhaegar Targaryen, the late Prince of Dragonstone</Text>
            </Card>
        </Card>
    </Story>
</Preview>

> **Note**: Shadow on hovering will appear only if you pass the `onClick` prop to it.

        <Text>Vertical</Text>
    </Card>
