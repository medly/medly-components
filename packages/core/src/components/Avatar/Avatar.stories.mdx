# Avatar

The `Avatar` component displays profile pictures, images, icons or initials, and text representing people or other entities.

## Color variations

You can either change the theme or pass the colors as props.

<Preview withToolbar>
    <Div>
        <Avatar
            sizes="L"
            textColor={defaultTheme.colors.green[500]}
            bgColor={defaultTheme.colors.green[100]}
            hoverBgColor={defaultTheme.colors.green[400]}
            hoverTextColor={defaultTheme.colors.white}
            hoverImgShadowColor="rgba(96, 120, 144, 0.35)"
            hoverTextShadowColor="rgba(0, 128, 0, 0.35)"
        >
            <Text>KP</Text>
        </Avatar>
        <Avatar
            sizes="L"
            textColor={defaultTheme.colors.blush[500]}
            bgColor={defaultTheme.colors.blush[100]}
            hoverBgColor={defaultTheme.colors.blush[400]}
            hoverTextColor={defaultTheme.colors.white}
            hoverImgShadowColor="rgba(96, 120, 144, 0.35)"
            hoverTextShadowColor="rgba(242, 113, 33, 0.35)"
        >
            <Text>KP</Text>
        </Avatar>
    </Div>
</Preview>
