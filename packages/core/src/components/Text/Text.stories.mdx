## Text with different Variants

You can use the font variant to decide the font style (`font-size`,`font-weight`,`line-height`,`letter-spacing`) as well as the `html tag`. If you use the `textWeight` prop as `Strong`|`ExtraStrong`, then both varaints `body1` and `body2` will use the `strong` html tag.

| Variant | Html Tag |
| ------- | -------- |
| h1      | h1       |
| h2      | h2       |
| h3      | h3       |
| h4      | h4       |
| h5      | h5       |
| body1   | span     |
| body2   | span     |
