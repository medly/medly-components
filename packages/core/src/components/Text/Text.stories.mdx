### Text with different Variants

Use font variant to decide the font style(`font-size`,`font-weight`,`line-height`,`letter-spacing`) and `html tag` also. If you have used `textWeight` prop as `Strong`|`ExtraStrong`, then both `body1` and `body2` variant will use `strong` html tag.

| Variant | Html Tag |
| ------- | -------- |
| h1      | h1       |
| h2      | h2       |
| h3      | h3       |
| h4      | h4       |
| h5      | h5       |
| body1   | span     |
| body2   | span     |
