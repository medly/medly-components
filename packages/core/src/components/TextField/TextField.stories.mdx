# TextField

You can use the basic `TextField` component to get user input. It is often used in conjunction with other form elements.

<Preview withToolbar>

## Multiline

To use a text area element instead of an input element, you must provide the `multiline` and `minRows` prop. The rows must be set to the number of rows you want to appear by default in the text area.

<Preview withToolbar>
    <TextField id="textarea" type="Name" variant="filled" multiline minRows={3} placeholder="Add Notes" />
    <TextField id="textarea" type="Name" variant="outlined" multiline minRows={3} placeholder="Add Notes" />
</Preview>

## Masking (beta-version)

To add the masking feature, you need to provide the `mask` prop. The feature currently supports only numeric input, so at the moment, you can only type numbers.

<Preview withToolbar>
    <TextField id="date" type="Name" variant="filled" label="Date" mask="DD / MM / YYYY" />
</Preview>

### With Prefix and Suffix

To use the label color, pass the className name prop to your prefix and suffix element, or you can directly use our icons.

### Form Validation

You can give a validator function to check the value and return the error messages. The HTML5 default error message will be displayed if your validator returns an empty string. We can create a generic react hook-like below to handle the onchange event.

<Preview>
    <Custom />
</Preview>

```tsx
type Result = [
    {
        value: string;
        onChange: (e: React.FormEvent<HTMLInputElement>) => void;
    },
    React.Dispatch<React.SetStateAction<string>>
];

export const useTextInput = (initialState = ''): Result => {
    const [value, setValue] = useState(initialState),
        onChange = useCallback((event: React.FormEvent) => setValue((event.target as HTMLInputElement).value), []);

    return [{ value, onChange }, setValue];
};
```

Additionally, you can use the above react hook in the form that is mentioned below.

```tsx
const emailValidation = (value: string) => {
    if (value === '') return 'Email is required';
    if (!value.includes('@')) return 'Email address should contain @';
};

export const Custom: React.FC = () => {
    const [email] = useTextInput(''),
        handleFormSubmit = useCallback((e: React.FormEvent) => e.preventDefault(), []);

    return (
        <Form onSubmit={handleFormSubmit}>
            <TextField type="email" label="Email" placeholder="Enter email" required {...email} validator={emailValidation} />
            <Button type="submit">Submit</Button>
        </Form>
    );
};
```
