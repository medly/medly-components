# Checkbox

It's a simple checkbox and a controlled react component. Handle the state as follows:

```tsx
const [isChecked, setChecked] = useState(false),
    onChangeHandler = (e: React.ChangeEvent<HTMLInputElement>) => setChecked(e.target.checked);

return <Checkbox label="Lorem Ipsum" checked={isChecked} onChange={onChangeHandler} />;
```

### Validations

1. This component calls the validator prop function with the latest value on blur, invalid, and change events. If you haven't passed the validator function, it will show the default html5 error validation message, if present.

```tsx
export const FormWithCheckbox: React.FC = props => {
    const [checked, setChecked] = useState<boolean>(),
        onChangeHandler = (e: React.ChangeEvent<HTMLInputElement>) => setChecked(e.target.checked),
        handleSubmit = (e: React.FormEvent) => e.preventDefault();
    return (
        <form onSubmit={handleSubmit}>
            <Checkbox required label="By clicking Submit, I agree lorem ipsum" checked={checked} onChange={onChangeHandler} {...props} />
            <Button type="submit">Submit</Button>
        </form>
    );
};
```

2. If you pass the validator function, it will be called with the latest selected value, so that, if required you can return the desired error message.


```tsx
validator={val => (!val ? 'Please accept the terms & conditions to continue.' : '')}
```
