# Drawer

A `Drawer` component is a modal anchored to the left or right edge of the screen. It consists of a Header, Content, and Footer. 

```typescript
<Drawer open={drawerState} onClose={changeDrawerState}>
    <Drawer.Header>Add Filters</Drawer.Header>
    <Drawer.Content>CONTENT</Drawer.Content>
    <Drawer.Footer>
        <Button variant="flat" onClick={onClearAll}>
            Clear All
        </Button>
    </Drawer.Footer>
</Drawer>
```

Shadow appears on the Header and Actions section if you scroll through the content.
