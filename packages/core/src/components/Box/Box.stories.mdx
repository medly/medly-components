# Box

The `Box` component serves as a wrapper component for most  CSS utility needs. It has customizable properties. By default it renders a `div`.
You can play with the props in the **Canvas** tab.

## Padding and margin

Define padding with `px` and `py`.

-   `px` - padding along x-axis.
-   `py` - padding along y-axis.

Define margin with `mx` and `my`.

-   `mx` - margin along x-axis.
-   `my` - margin along y-axis.

> **Note**: Values for padding and margin are defined as a `number` and then converted into `rem` using `(value * 0.25rem)`

<Preview>
    <Box px={4} py={4} mx={0} my={0} borderWidth={1} borderColor="tomato">
        Box with padding
    </Box>
    <Box mx={8} my={4} px={0} py={0} borderWidth={1} borderColor="tomato">
        Box with margin
    </Box>
</Preview>

## Color and background color

Provide a color and background color to a box component. Use `color` and `bg` respectively.

<Preview>
    <Box color="white" bg="black" mx={2}>
        Box with white color and black background
    </Box>
    <Box color="white" bg={defaultTheme.colors.blue[500]} mx={2}>
        Box with white color and blue background
    </Box>
</Preview>

## Height and width

Define the `height` and `width` as any CSS (rem | px | percent) unit.

<Preview>
    <Box color="white" bg="black" mx={2} height="20rem" width="40rem">
        Box with 20rem height and 40rem width
    </Box>
    <Box color="white" bg="black" mx={2} height="10rem" width="20rem">
        Box with 10rem height and 20rem width
    </Box>
</Preview>

## Border

Add a border using `borderWidth` and `borderColor`. Additionally, to include a border radius use `borderRadius`.

> **Note**: Add a border only when both `borderWidth` and `borderColor` are provided.

> Value for borderWidth and borderRadius are defined as a `number` and then converted into `rem` using `(value * 0.25rem)`

<Preview>
    <Box borderColor={defaultTheme.colors.grey[500]} borderWidth={1} mx={2} borderRadius={0}>
        Box with grey border
    </Box>
    <Box borderColor={defaultTheme.colors.grey[500]} borderWidth={1} mx={2} borderRadius={4}>
        Box with border radius of 4
    </Box>
</Preview>

## Font variant

It defines the font size to use inside the box component.

> **Note** The font-variant is only applied if the child is a text value.

<Preview>
    <Box mx={2} fontVariant="h1" borderColor={defaultTheme.colors.grey[500]} borderWidth={1}>
        Box with h1 font
    </Box>
    <Box mx={2} fontVariant="h4" borderColor={defaultTheme.colors.grey[500]} borderWidth={1}>
        Box with h4 font
    </Box>
</Preview>

## Shadow

Four `shadowSize` sizes are available: `S` | `M` | `L` | `XL`

To customize shadow use: `shadowColor`

<Preview>
    <Box mx={2} shadowSize="S">
        Box with small shadow
    </Box>
    <Box mx={2} shadowSize="M">
        Box with medium shadow
    </Box>
    <Box mx={2} shadowSize="L">
        Box with large shadow
    </Box>
    <Box mx={2} shadowSize="XL">
        Box with extra large shadow
    </Box>
</Preview>
