# Modal

Basic modal component. You can play with the component in the canvas tab.

Modal consists of three sections Header, Content and Actions. All of these you can directly use from Modal component only like bellow . . .

```tsx
import { Modal } from '@medly-components/core';
import React, { useState } from 'react';

export const DummyComponent: React.FC = React.memo(() => {
    const [state, updateState] = useState(false);

    const toggleModal = () => updateState(val => !val);

    return (
        <Modal open={state} onCloseModal={toggleModal}>
            <Modal.Header>Add User</Modal.Header>
            <Modal.Content>
                <Input id="name-input" type="text" fullWidth label="Name" placeholder="Enter your Name" />
            </Modal.Content>
            <Modal.Actions>
                <Button variant="outlined">Add User</Button>
            </Modal.Actions>
        </Modal>
    );
});

DummyComponent.displayName = 'DummyComponent';
```

## Shadow Effect

Shadow will apear on the Header & Actions section if you scroll the content.

### Keyboard Support

It also supports keyboard navigation when it is in focus

| Key | Function        |
| --- | --------------- |
| Esc | Close the modal |
