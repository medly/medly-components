# Toast

The `Toast` component is a popup window that display a message to a user. You can use the toast component to give feedback to users after an action takes place. It can be a success message, error message, warning message, and custom or info message.

## How to use

You have to add `<ToastContainer/>` in your app at the root level and to raise the toast use the `addToast` function. For more info check `ToastContainer` docs.

<Props of={ThemeInterface} />

## Custom Header and Message component

You can pass the custom header and message components.

```tsx
<Toast
    id="1"
    variant="success"
    header={<Text textVariant="h4">Lorem Ipsum is simply dummy text.</Text>}
    message={<Text textVariant="body2">Lorem Ipsum is simply dummy text.</Text>}
></Toast>;

OR;

addToast({
    variant: 'success',
    header={<Text textVariant="h4">Lorem Ipsum is simply dummy text.</Text>},
    message={<Text textVariant="body2">Lorem Ipsum is simply dummy text.</Text>}
});
```
