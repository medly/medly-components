# Header

The `Header` component an important element of a website’s design. It’s the first impression of the website. It provides useful links to other areas of the website that the user may want to visit.

<Preview mdxSource="">
    <Story name="Header">
        {() => {
            const [activeTab, setActiveTab] = useState('dashboard');
            const isNotMobile = useMediaQuery(breakpoints(defaultTheme.breakpoints).up('M'));
            return (
                <div style={{ overflow: 'hidden', height: '400px' }}>
                    <Header>
                        <Logo showName={isNotMobile} />
                        <Header.Nav>
                            <Header.NavItem isActive={activeTab === 'dashboard'} onClick={() => setActiveTab('dashboard')}>

Given below is a basic navigation setup. The Logo/Icons shown in the illustration can be easily customized as needed.

```tsx
import Header from '@medly-components/layout/Header';
import Logo from '@medly-components/core/Logo';
import { HomeIcon, PillIcon, LockIcon } from '@medly-components/icons';
import { defaultTheme } from '@medly-components/theme';

<Header>
    <Logo showName={isDesktop} />
    <Header.Nav>
        <Header.NavItem isActive>
            <HomeIcon size="S" iconColor={defaultTheme.colors.white} />
            My Dashboard
        </Header.NavItem>
        <Header.NavItem>
            <PillIcon size="S" iconColor={defaultTheme.colors.white} />
            Prescriptions
        </Header.NavItem>
        <Header.NavItem>
            <LockIcon size="S" iconColor={defaultTheme.colors.white} />
            Security
        </Header.NavItem>
    </Header.Nav>
    <div>Logout</div>
</Header>;
```
