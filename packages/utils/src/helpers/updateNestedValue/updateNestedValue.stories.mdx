# updateNestedValue

The `updateNestedValue` function returns a new object that copies the structure of an input object while replacing a nested object value with another.

Any existing, non-primitive nested values (except objects) will retain their original references in the returned value (i.e. no deep copy performed).

All properties referenced in the nested object path for replacement are expected to exist.

### Parameters

- The `obj` is the nested object to operate.
- The `key` is a period-delimited path to a nested property (e.g. `a.b.c`).
- The `value` is an object to replace the value at the nested property path denoted by `key`.

### Example

One practical use within the context of components is to override thematic style values.

```tsx
<ThemeProvider
    theme={updateNestedValue(defaultTheme, 'button.flat.textColor', {
        default: green,
        disabled: red,
        hovered: lightgreen,
        pressed: lightgreen,
    })}
>
    <Button variant="flat">Click Me!</Button>
</ThemeProvider>;
```

**Note**: Keep in mind that the replacement does not preserve any property values that may have existed previously. 
It might be useful to consider the spread operator in forming the replacement value when only a subset of property values is of interest to be updated.

```ts
const updatedTheme = updateNestedValue(defaultTheme, 'button.flat.textColor', {
    ...defaultTheme.button.flat.textColor,
    disabled: gray,
});
```
