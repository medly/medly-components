# media

The `media` function creates a media query block in any styled component.

### Example

```tsx
${
media(breakpoints(theme.breakpoints).up('M'))`
            display: none;
        `;
}
```

The above snippet will return the below css:

```css
@media (min-width: 768px) {
    display: none;
}
```
