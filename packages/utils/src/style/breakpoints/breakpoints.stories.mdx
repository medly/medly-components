# breakpoints

The `breakpoints` function creates media queries based on the given breakpoints: (`S` | `M` | `L` | `XL`). You need to pass the theme's breakpoints object to it, and then it will return the object with the functions given below:

| Name     | Param                              | Return                                                                                                             |
| -------- | ---------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| up       | key: Breakpoint                    | `@media (min-width: ${bp[key].min}px)`                                                                             |
| down     | key: Breakpoint                    | `@media (max-width: ${bp[key].max}px)`                                                                             |
| between  | start: Breakpoint, end: Breakpoint | `@media (min-width: ${bp[start].min}px) and (max-width: ${bp[end].max}px)`                                         |
| only     | key: Breakpoint                    | between(key, key)                                                                                                  |
| multiple | sizes: Breakpoint[]                | `@media ${sizesAsArray.map(size =>`(min-width: ${bp[size].min}px) and (max-width: ${bp[size].max}px)`).join(',')}` |

### Example

`breakpoints(theme.breakpoints).up('M');` returns `@media (min-width: 768px)`

You must use it with the `media` function provided by `@medly-components/utils` like below:

```tsx
${
media(breakpoints(theme.breakpoints).up('M'))`
            display: none;
        `;
}
```

You can use the above snippet in any styled component, it returns the css given below:

```css
@media (min-width: 768px) {
    display: none;
}
```
