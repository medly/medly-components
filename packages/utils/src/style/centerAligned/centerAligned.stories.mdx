# centerAligned

`centerAligned` can be used to add css for center aligning the child elements.

#### Parameters

`display` - It can be either `flex` or `inline-flex`.

**Note: The default value for dispaly is `inline-flex`**

#### Example

You should use it when you need to center a child item.

```css
import styled from 'styled-components';
import { centerAligned } from '@medly-components/utils';

const Element = styled('div')`
    ${centerAligned()}
`;
```

The above snippet will return below css:

```css
display: inline-flex;
justify-content: center;
align-items: center;
```
