# centerAligned

You can use `centerAligned` to add CSS to center align the child elements.

### Parameters

`display` - It can be either, `flex` or `inline-flex`.

**Note: The default value dispaly is `inline-flex`**

### Example

You must use it when you need to center a child item.

```css
import styled from 'styled-components';
import { centerAligned } from '@medly-components/utils';

const Element = styled('div')`
    ${centerAligned()}
`;
```

The above snippet will return the below css:

```css
display: inline-flex;
justify-content: center;
align-items: center;
```
