# getFontStyle

The `getFontStyle` is a  styled component helper that generates font-related styles based on the passed theme. It accepts a specific font-variant or font-weight to influence the generated output.

### Example

The simple usage without any arguments must fall back on using configured defaults for font-related styles.

```
const plainParagraph = styled('p')`
    ${getFontStyle}
`;
```

Any specified font-variant or font-weight must take precedence in applying.

```
const seriousParagraph = styled('p')`
    ${({ theme }) => getFontStyle({ theme, fontWeight: 'ExtraStrong' })}
`;
```
