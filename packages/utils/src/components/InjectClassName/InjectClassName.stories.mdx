# InjectClassName

`InjectClassName` can be used to add a className to all its children.

#### Parameters

It takes only one parameter `className` which is of type string.

#### Example

```tsx
import { InjectClassName } from '@medly-components/utils';

const InjectClassNameComponent: React.FC = () => {
    return (
        <InjectClassName className="box">
            <div>Box 1</div>
            <div>Box 2</div>
        </InjectClassName>
    );
};
```

The above code snippet will return below jsx:

```jsx
<div className="box">Box 1</div>
<div className="box">Box 2</div>
```
