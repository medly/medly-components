# InjectClassName

You can use the `InjectClassName` to create styles for child elements without providing the HTML tag to the styled-components.

### Example

```tsx
import { defaultTheme } from '@medly-components/theme';
import { InjectClassName } from '@medly-components/utils';
import styled from 'styled-components';
import { SvgLoaderProps } from './types';

export const SvgLoaderStyled = styled(InjectClassName)<SvgLoaderProps>`
    font-size: ${({ theme, size }) => theme.loader.sizes[size!]};
    * {
        fill: ${({ theme, color }) => color || theme.loader.defaultColor};
    }
`;

SvgLoaderStyled.displayName = 'SvgLoader';
SvgLoaderStyled.defaultProps = {
    size: 'S',
    theme: defaultTheme
};
```

You can use the above `SvgLoader` to style any loader given below:

```jsx
<SvgLoader>
    <ConcentricCircleLoaderSvg />
</SvgLoader>
```

**Note**: The `SvgLoader` style applies directly to `ConcentricCircleLoaderSvg`.
