# useOuterClickNotifier

This hook triggers the provided function on certain events affecting the ref passed in.

More specifically, the function triggers when:

- A click occurs anywhere outside the ref target
- Whenever the <kbd>ESC</kbd> key is pressed

#### Parameters

- `onOuterClick` is a function to be triggered
- `innerRef` is a ref to some element

#### Example

The following demonstrates a simple use of this hook to clear the contents of
a text input after a user finishes typing in it and meets the trigger conditions
outlined above.

```tsx
const AutoClearTextInput: React.FC = () => {
    const ref = useRef(null);
    useOuterClickNotifier(() => {
        if (ref.current) {
            ref.current.value = '';
        }
    }, ref);
    return <input ref={ref} type='text' />;
};
```
